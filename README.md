# ## Machine Learning Internship - Codveda Technology
This repository contains my comprehensive project submissions for the Machine Learning Internship at *Codveda Technology*. The projects are organized into three levels, progressing from data preprocessing and basic regression to advanced deep learning.
## ## 📁 Level 1: Basic
*Datasets:* Housing Prices Dataset
 * *Task 1: Data Preprocessing*: Cleaned raw data by handling missing values and encoding categorical variables. Standardized numerical features and split data into training/testing sets.
 * *Task 2: Simple Linear Regression: Developed a model to predict continuous variables (house prices). Evaluated performance using **R-squared* and *Mean Squared Error (MSE)*.
## ## 📁 Level-2: Intermediate
*Datasets:* Mall Customer Segmentation
 * *Task 2: Decision Trees*: Implemented a Decision Tree Regressor to analyze categorical outcomes. Visualized the tree structure to interpret decision-making nodes.
 * *Task 3: K-Means Clustering: Grouped unlabeled customer data into distinct clusters. Utilized the **Elbow Method* to determine the optimal number of clusters and visualized results with scatter plots.
## ## 📁 Level-3: Advanced
*Datasets:* Breast Cancer Diagnostic (Built-in) & MNIST Digits (TensorFlow)
 * *Task 2: Support Vector Machine (SVM): Built a binary classification model for medical diagnostics. Compared performance between **Linear* and *RBF kernels* and visualized the results via a confusion matrix.
 * *Task 3: Neural Networks: Designed a feed-forward neural network using **TensorFlow/Keras* for digit classification. Monitored training through backpropagation and visualized accuracy and loss curves.
## ## 🛠️ Tools & Technologies
 * *Language*: Python
 * *Libraries*: Pandas, Scikit-Learn, TensorFlow, Keras, Matplotlib, Seaborn
 * *Environment*: Google Colab
